Step 1 — apt packages: Installs nmap, python3, xsltproc, nikto, smbclient, enum4linux, snmpwalk, redis-tools, curl, wget, git, dnsutils, netcat, and jq. Skips anything already present.
Step 2 — Python packages: Installs pyyaml (required for playbook triage scoring), requests, and lxml via pip with --break-system-packages for Kali compatibility.
Step 3 — External tools not in apt:

ffuf — auto-detects latest release from GitHub API and downloads the binary
gobuster — via apt
whatweb — via apt or gem fallback
netexec (nxc) — via pip, falls back to pipx, warns if both fail
wpscan — via gem
vulners.nse — downloads directly from the vulnersCom GitHub repo and updates the nmap script DB
SecLists — via apt or git clone fallback to /usr/share/seclists

Step 4 — Downloads your two GitHub files using the raw URL, strips the .txt extension on save, and places everything in /opt/oscp-toolkit/. Creates a oscp-scan symlink in /usr/local/bin so you can run it from anywhere.
Step 5 — Syntax check on the downloaded launcher with python3 -m py_compile.
Step 6 — Creates /opt/oscp-toolkit/scan_runs/ as the working output directory.
Step 7 — Final tool summary with ✓/✗ for every tool the launcher uses.
# Run
sudo python3 interactive_nmap_launcher_v4.py
